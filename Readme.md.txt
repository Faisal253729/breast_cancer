# Advanced ANN for Breast Cancer Classification

This project implements an **Advanced ANN** to classify breast cancer using the **Breast Cancer dataset from scikit-learn**.  

## Features
- Multi-layer ANN with ReLU and regularization
- Training & validation metrics
- Streamlit web app for interactive prediction

## Files
- `streamlit_app.py` : Streamlit app
- `ann_model.h5` : Trained ANN model
- `scaler.save` : Scaler for input preprocessing
- `requirements.txt` : Dependencies
- `.gitignore` : Git ignore file
